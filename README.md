# IST to PST Converter (HTML)

This is a simple HTML-based tool to convert Indian Standard Time (IST) to Pacific Standard Time (PST).

## How to Use
1. Open `index.html` in any modern web browser.
2. Enter IST time in the format `YYYY-MM-DD HH:MM` (e.g., `2025-11-11 19:10`).
3. Click **Convert** to see the PST equivalent.
